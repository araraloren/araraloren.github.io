readelf用法小记

readelf是一个用来读取elf信息的工具，可以读取elf各分节的信息。

对于文件elffile

  • readelf -S elffile

    列出elffile中的所有节头, [下标] 节名 类型 地址 偏移 大小 符号表大小 标志 链接 信息 字节对齐大小

  • readelf [-x|p|R] [下标|节名] elffile

    读取当前节的内容

  • readelf -h elffile

    显示当前文件的elf文件头部

  • readelf -l elffile

    显示当前文件的程序头部信息

  • readelf -t elffile

    显示分节的详细信息

  • readelf -s elffile

    显示符号表

关于C的字符串

  • under linux

  • const char* str = “1234354”;

    存在于.rodata中

  • char str[] = “123456”;

    存在于.text中，之后被会复制到栈上
